# 🎓 Responsive Duplicate Website – A.M. Reddy College

## 📌 Project Overview
This project is a **responsive duplicate website** of A.M. Reddy College, built using core web technologies. It replicates the structure and design of a college website while ensuring compatibility across different devices such as desktops, tablets, and smartphones.

The main objective of this project is to practice **frontend development skills** and understand how real-world educational websites are structured and designed.

---

## 🚀 Features
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean and modern UI
- 🏫 College-style layout (Home, About, Courses, Contact, etc.)
- ⚡ Fast loading and simple navigation
- 🧩 Well-structured sections

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript
